Simple script that is renaming all files in the folder and numerates them.

## Installation 
```bash
git clone https://github.com/cxnexion/bulk-rename
cd bulk-rename
npm install
npm start
```

## Usage

**Important: Don't use with files that have more than 1 extension!!! Example: importantFile.test.ts , it will be renamed to newName.ts!!!**

npm start <absolute_directory_path> <name>
